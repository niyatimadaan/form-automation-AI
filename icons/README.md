# Icons

Extension icons for Form Autofill Assistant.

The icon.svg file contains the source design. To generate PNG files at different sizes:

1. Use an SVG to PNG converter or image editor
2. Export at 16x16, 48x48, and 128x128 pixels
3. Name them icon16.png, icon48.png, and icon128.png

For development, you can use the SVG directly or create simple placeholder PNGs.

The icon design shows:
- Blue background representing forms
- White horizontal lines representing form fields
- Green checkmark indicating autofill completion
